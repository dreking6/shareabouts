About PlanShare

PlanShare is a web app designed to gather constructive public input in a social and engaging framework that allows all stakeholders to communicate their plan ideas effectively. The PlanShare Interactive map tool is designed to enable all participants to be fully engaged in planning process from the start to contribute their quota to the development of a community or state. As a participant various tasks can be realized using the web app. Due to diversity in Urban and rural needs, PlanShare adopts the use of templates. The Pilot version of PlanShare adopts 2 default templates. New templates that can adapt to locational and organizational needs can be designed based on request.
